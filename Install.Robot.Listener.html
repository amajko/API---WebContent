In order to set your ROBOT instance to auto-report to Guardian:

  1. Download this GwListener.jar to your directories at:
    • Windows: %QSI_HARNESS_HOME%\bin
      • or
    • Linux: $QSI_HARNESS_HOME/bin
  2. Set this environment variable at:
    • Windows, Control Panel, System, Advanced, Environment Variables, System Variables:
      "HARNESS_ROBOT_LISTENER"
      to
      "com.nokia.dddeyet.listener.client.ReportListener:dchiecapp06.hq.navteq.com:8080"

      • or
    • Linux: "export HARNESS_ROBOT_LISTENER=com.nokia.dddeyet.listener.client.ReportListener:dchiecapp06.hq.navteq.com:8080"
  3. In order to temporarily shut off any communication to Guardian,
    add this argument to runtests.bat "-M guardian:ignore"
    or
    in RIDE gui:
  4. In order to add the TestAutomation listener from TA department,
    add this argument to runtests.bat
    "--listener com.navteq.ta.listener.robot.ReportListener:10.228.50.102:7080:3DIT:[ANY_EPIC_NAME]:[ANY_TEST_PURPOSE]"
    or
    in RIDE gui:
